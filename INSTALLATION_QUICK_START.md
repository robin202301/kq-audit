# AuditSystem-Win 快速安装指南

## 快速安装步骤

### 1. 下载安装程序
从发布页面下载最新版本的 `KQQ-Audit-System Setup x.x.x.exe`

### 2. 运行安装程序
1. 双击安装程序文件
2. 如果出现安全警告，点击"更多信息" → "仍要运行"
3. 选择安装语言（默认中文）

### 3. 安装选项
- **安装目录**: 建议使用默认路径 `C:\Program Files\KQQ-Audit-System\`
- **开始菜单**: 创建快捷方式（推荐）
- **桌面图标**: 创建桌面快捷方式（可选）

### 4. 完成安装
点击"安装"按钮，等待安装完成。

## 首次运行配置

### 1. 启动应用程序
- 从开始菜单或桌面快捷方式启动
- 或直接运行安装目录下的 `KQQ-Audit-System.exe`

### 2. 初始设置
1. **同意用户协议**: 阅读并接受许可条款
2. **选择工作目录**: 设置项目文件保存位置
3. **模板验证**: 系统会自动验证模板文件完整性

### 3. 创建第一个项目
1. 点击"新建项目"按钮
2. 输入项目名称和描述
3. 选择审计类型和时间范围
4. 开始使用6阶段审计工作流

## 系统要求检查清单

### 最低要求
- [ ] Windows 7 或更高版本（64位）
- [ ] 2GB 内存
- [ ] 500MB 可用磁盘空间
- [ ] .NET Framework 4.5 或更高版本

### 推荐配置
- [ ] Windows 10/11（64位）
- [ ] 4GB 内存或更高
- [ ] 1GB 可用磁盘空间
- [ ] SSD 硬盘以获得更好性能

## 常见问题快速解决

### ❌ 安装被阻止
**问题**: Windows SmartScreen 阻止安装
**解决**:
1. 右键点击安装程序 → "属性"
2. 勾选"解除锁定"（如果显示）
3. 再次运行安装程序

### ❌ 缺少运行库
**问题**: 提示缺少 `VCRUNTIME140.dll`
**解决**:
1. 下载并安装 [VC++ Redistributable](https://aka.ms/vs/17/release/vc_redist.x64.exe)
2. 重新运行安装程序

### ❌ 无法启动
**问题**: 应用程序启动后立即关闭
**解决**:
1. 以管理员身份运行
2. 检查事件查看器中的错误日志
3. 重新安装应用程序

### ❌ Word模板无法使用
**问题**: 生成Word文档时提示模板文件丢失
**解决**:
1. 检查 `%APPDATA%\KQQ-Audit-System\resources\templates\` 目录
2. 重新安装应用程序
3. 联系技术支持

## 数据备份和迁移

### 备份用户数据
```
# 备份配置文件
%APPDATA%\KQQ-Audit-System\config.json

# 备份数据库
%APPDATA%\KQQ-Audit-System\audit.db

# 备份用户模板
%USERPROFILE%\Documents\KQQ-Audit-System\Templates\
```

### 迁移到新电脑
1. 在新电脑安装 AuditSystem-Win
2. 将备份文件复制到对应位置
3. 启动应用程序，数据会自动加载

## 卸载和清理

### 完全卸载
1. 控制面板 → 程序和功能
2. 选择"KQQ-Audit-System"
3. 点击"卸载"
4. 删除剩余的用户数据（如果需要）

### 保留数据卸载
卸载前备份上述数据目录，重新安装后恢复。
